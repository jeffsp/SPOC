# NAME

spoc_spoc2text - Convert a SPOC format file to text

# USAGE

spoc_spoc2text [*options*] *filename.spoc* *filename.txt*

# DESCRIPTION

Convert a SPOC format file to text

# OPTIONS

\-\-help, -h
:   Get help

\-\-verbose, -v
:   Set verbose mode ON

\-\-version, -e
:   Print version information and exit
