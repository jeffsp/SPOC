# NAME

spoc_las2spoc - Convert a LAS format file to a SPOC format file

# USAGE

spoc_las2spoc [*options*] *filename.las* *filename.spoc*

# DESCRIPTION

Convert a LAS format file to a SPOC format file


# OPTIONS

\-\-help, -h
:   Get help

\-\-verbose, -v
:   Set verbose mode ON

\-\-version, -e
:   Print version information and exit
