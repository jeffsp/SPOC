# NAME

spoc_decompress - Decompress a spoc file

# USAGE

spoc_decompress [*options*] [*input_filename*] [*output_filename*]

# DESCRIPTION

Decompress a spoc file.

# OPTIONS

\-\-help, -h
:   Get help

\-\-verbose, -v
:   Set verbose mode ON

\-\-version, -e
:   Print version information and exit
