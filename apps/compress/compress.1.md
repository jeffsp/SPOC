# NAME

spoc_compress - Compress a spoc file

# USAGE

spoc_compress [*options*] [*input_filename*] [*output_filename*]

# DESCRIPTION

Compress a spoc file.

# OPTIONS

\-\-help, -h
:   Get help

\-\-verbose, -v
:   Set verbose mode ON

\-\-version, -e
:   Print version information and exit
