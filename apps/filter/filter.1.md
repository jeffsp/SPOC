# NAME

spoc\_filter - Filter out points from a SPOC file

# USAGE

spoc\_filter [*options*] [*input_filename*] [*output_filename*]

# DESCRIPTION

Filter out points from a SPOC file

# OPTIONS

\-\-help, -h
:   Get help

\-\-verbose, -v
:   Set verbose mode ON

\-\-version, -e
:   Print version information and exit
