# NAME

spoc_text2spoc - Convert a text format file to a SPOC format file

# USAGE

spoc_text2spoc [*options*] *filename.txt* *filename.spoc*

# DESCRIPTION

Convert a text format file to a SPOC format file

# OPTIONS

\-\-help, -h
:   Get help

\-\-verbose, -v
:   Set verbose mode ON

\-\-version, -e
:   Print version information and exit
